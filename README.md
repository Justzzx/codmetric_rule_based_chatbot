# Rule-Based Chatbot

## Description
A simple rule-based chatbot built using Python. The chatbot responds to greetings and common user queries using predefined rules and conditional statements.

## Features
- Greeting responses
- AI-related responses
- Name identification
- Help command
- Exit command

## Technology Used
- Python

## How to Run
1. Open terminal.
2. Navigate to project folder.
3. Run:

python rulebasedchatbot.py
